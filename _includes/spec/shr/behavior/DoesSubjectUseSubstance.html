DoesSubjectUseSubstance

Records if the subject uses a particular substance or class of substances
Based On: DidExposureOccur The likelihood the exposure happened, e.g. exposed to asbestos in his workplace (using Setting). If missing, it is assumed the exposure did occur.
Value: Coding from QualitativeLikelihoodVS Coding of a concept, drawn from a controlled vocabulary. Includes the vocabulary and version, if applicable. May include a display text, and a descriptor expressing the intended interpretation of the code
