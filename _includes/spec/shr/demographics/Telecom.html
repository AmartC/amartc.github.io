Telecom

An electronic means of contacting an organization or individual C2986441
Value: TelecomNumberOrAddress A user name or other identifier on a telecommunication network, such as a telephone number (including country code and extension, if necessary), email address, or SkypeID
TelecomType 1 The type or channel of the communication, such as email or telephone. If an messaging app, such as Skype, name the application using string
TelecomQualifier 0 or more Provides more information about the type and uses of the telecom address
EffectiveTimePeriod optional The date and time span for which something is active, valid, or in force.
