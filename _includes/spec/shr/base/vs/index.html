Base Value Sets

MissingValueReasonVS

Reasons that a value associated with a test or other observation is missing.

Code Display CodeSystem
missing_unanswered The source was asked but declined to respond to the question, or the question was left unanswered. MissingValueReasonVS
missing_na The question or data element is not applicable, or based on other answers, no answer to this question is required. MissingValueReasonVS
missing_redacted The information has been redacted. MissingValueReasonVS
missing_no_clue The reason the information is not present is not known. MissingValueReasonVS
missing_cancelled The request or order to collect this information was cancelled. MissingValueReasonVS
missing_collection Missing due to a problem collecting, identifying, or locating the specimen, including patient refusal or unable to provide specimen MissingValueReasonVS
missing_specimen Missing due to a problem with the specimen, e.g. contamination, clotting, improper tube type, improper storage, too small, etc. MissingValueReasonVS
missing_malfunction Missing due to instrument malfunction. MissingValueReasonVS
