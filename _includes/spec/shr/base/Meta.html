Meta

Contains elements common to all entries, describing the identification, timing, attribution, and other properties of the entry. C1708992
EntryID 1 A unique, persistent, permanent identifier for an entry in a health record
EntryType 1 or more The SHR data element identifier, for example, http://standardhealthrecord.org/allergy/SubstanceRisk. The EntryType array includes the entire 'based on' hierarchy to enable searching for classes of things, for example, searching for all vital signs, or all types of behavioral observations.
Version optional A string identifying the particular of form of something (such as a code system or software product) that is different in some way from another form of the same thing.
Informant 0 or more The person or entity that provided the information in the entry, as distinct from the actor creating the entry, e.g. the subject (patient), medical professional, family member, device or software program.
Author 0 or more The person or organization who created the entry and is responsible for (and may certify) its content.
AssociatedEncounter 0 or more The encounter or episode of care that led to creation of this entry
DateEntered 1 The point in time when the information was recorded in the system of record.
DateUpdated 0 or more A date that the entry was changed
Language optional A human language, spoken or written
SecurityLabel (TBD) 0 or more
Tags (TBD) 0 or more
