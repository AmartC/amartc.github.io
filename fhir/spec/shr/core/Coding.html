Coding

Coding of a concept, drawn from a controlled vocabulary. Includes the vocabulary and version, if applicable. May include a display text, and a descriptor expressing the intended interpretation of the code. C0805701
Value: code
Vocabulary optional A formal terminology system.
VocabularyVersion optional The version of the vocabulary being used, if applicable.
DisplayText optional A string meant for reading by a person, usually accompanying a code.
CodingQualifier 0 or more A description of the purpose or origin of the code that assists in the interpretation of the code.
