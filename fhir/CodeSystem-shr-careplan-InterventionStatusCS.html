SHR InterventionStatusCS CodeSystem

Value indicating whether an activity is currently being acted upon, represents future intentions or is now a historical record. From HL7 V3 Reference Information Model.

This code system http://standardhealthrecord.org/shr/careplan/cs/InterventionStatusCS defines the following codes:
